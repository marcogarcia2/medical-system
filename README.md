## Como utilizar
Para acessar o contêiner:
```bash
docker compose up -d
docker exec -it django_app bash
```

Para iniciar o servidor:
```bash
python manage.py runserver 0.0.0.0:8000
```

Acesse em seu navegador o site:

http://127.0.0.1:8000/
